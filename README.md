# iPlayground 主辦模擬器 🎪

肉鴿抽卡小遊戲：你是 iPlayground 總召，距離活動日 9 週。轉輪盤邀講者、招募工作人員、宣傳售票、看準時機公布議程——目標把報名衝到 **350 人滿員**。

> 粉絲作品，非官方。資料來自 [iplayground/SessionData](https://github.com/iplayground/SessionData)（2026）。
> 輪盤引擎源自前作 [講者抽抽樂](https://iplayground-speaker-wheel.vercel.app/)。

## 玩法

- **雙語**：右上角切換 中文 / English（講者名、講題、卡面全對應 SessionData 雙語資料）
- **週回合制**：9 週，每週基礎 **2 行動點**——一周目失敗後籌委會追加預算，之後每輪 **3 點**（工作人員每 3 位再 +1，上限 5）
- **🎤 邀請講者**（每週最多 3 次）：轉輪盤抽講者卡——一般講者／Workshop（紫）／Lightning，不分等級
- **🙋 招募工作人員**：獨立卡池，純戰力，疊數量換行動點
- **📣 宣傳售票**：2d6 ＋ 已公布講者加成
- **📋 公布議程**：核心機制——公布後講者陣容**每週持續**帶來報名（一般講者每週 3~5、Workshop 2~4、Lightning 1~3，陣容越大乘數越高，×1.3 封頂）。越早公布吃越多週複利，但太早公布陣容小、每週數字低——抓好時機！6 大 tag 湊齊後議程效應永久 ×1.2
- **雙軌事件系統**：每週固定一則**主事件**（敘事完整、可有選項、影響大——颱風、流感、講者臨時取消、大神轉發、指定邀請、Apple 突發新框架、場地換廳、票價論戰……），結束後 50% 機率追加一則**輕事件快訊**（一句話、無選項、小影響）。難度會加權好壞事件比例
- **🎭 神秘講者來訪**（招牌主事件）：場景切成講者全黑剪影，四選一猜出是誰——猜對，該講者免費入列（不佔上限、不花行動點）；猜錯也會點亮公布正解，讓你認識他

### 勝利條件

一般講者 ≥3、Workshop ≥1、Lightning ≥1，報名 ≥200（350 滿員）。
結算 S/A/B/C 評分 = 報名人數 ＋ tag 多樣性 ×10 ＋ 國際講者 ×15。

一般講者湊滿 6 大主題分類（AI／架構／UI-UX／Swift／跨平台／產品）→ 議程效應永久 ×1.2。

## 開發

單檔 `index.html`，無相依、無 build。直接開檔案或任何靜態伺服器即可。

議程異動時更新卡池資料：

```bash
python3 update_data.py
```
