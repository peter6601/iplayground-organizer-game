# iPlayground 主辦模擬器 🎪

肉鴿抽卡小遊戲：你是 iPlayground 總召，距離活動日 9 週。轉輪盤邀講者、招募工作人員、宣傳售票、看準時機公布議程——目標把報名衝到 **350 人滿員**。

> 粉絲作品，非官方。資料來自 [iplayground/SessionData](https://github.com/iplayground/SessionData)（2026）。
> 輪盤引擎源自前作 [講者抽抽樂](https://iplayground-speaker-wheel.vercel.app/)。

## 玩法

- **週回合制**：9 週，每週 3 行動點（工作人員每 3 位 +1，上限 5）
- **🎤 邀請講者**（每週最多 3 次）：轉輪盤抽講者卡——SR 金框 40 分鐘主講、R 一般、N Lightning、W Workshop
- **🙋 招募工作人員**：獨立卡池，純戰力，疊數量換行動點
- **📣 宣傳售票**：2d6 ＋ 已公布講者加成
- **📋 公布議程**：核心機制——講者不公布不會帶來報名！一次公布越多張話題加成越高（×1.5 封頂），但第 8 週後才公布會打折
- **每週事件卡**：颱風、流感、講者臨時取消（有備援講者可救場）、大神轉發、指定邀請……

### 勝利條件

一般講者 ≥3、Workshop ≥1、Lightning ≥1，報名 ≥200（350 滿員）。
結算 S/A/B/C 評分 = 報名人數 ＋ tag 多樣性 ×10 ＋ 國際講者 ×15。

一般講者湊滿 6 大主題分類（AI／架構／UI-UX／Swift／跨平台／產品）→ 報名一次性 +30。

## 開發

單檔 `index.html`，無相依、無 build。直接開檔案或任何靜態伺服器即可。

議程異動時更新卡池資料：

```bash
python3 update_data.py
```
